QString QFileDialog::selectedFile () const

Use selectedFiles() instead.

For example, if you have code like

 QString selected = dialog->selectedFile();

you can rewrite it as

  QStringList files = dialog->selectedFiles();
  QString selected;
  if (!files.isEmpty())
      selected = files[0];
