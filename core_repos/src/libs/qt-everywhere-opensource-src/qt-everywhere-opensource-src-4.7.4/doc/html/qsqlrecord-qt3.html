Qt 3 Support Members for QSqlRecord

The following class members are part of the Qt 3 support layer. They are provided to help you port old code to Qt 4. We advise against using them in new code.

QString QSqlRecord::toString ( const QString & prefix = QString(), const QString & sep = QLatin1String( "," ) ) const

Returns a list of all the record's field names as a string separated by sep.

In the unlikely event that you used this function in Qt 3, you can simulate it using the rest of the QSqlRecord public API.

QStringList QSqlRecord::toStringList ( const QString & prefix = QString() ) const

Returns a list of all the record's field names, each having the prefix prefix.

In the unlikely event that you used this function in Qt 3, you can simulate it using the rest of the QSqlRecord public API.
