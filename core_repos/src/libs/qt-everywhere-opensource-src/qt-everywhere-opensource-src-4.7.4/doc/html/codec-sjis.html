Shift-JIS Text Codec

The Shift-JIS codec provides conversion to and from Shift-JIS, an encoding of JIS X 0201 Latin, JIS X 0201 Kana and JIS X 0208.

The environment variable UNICODEMAP_JP can be used to fine-tune the codec. The ISO 2022-JP (JIS) Text Codec documentation describes how to use this variable.
